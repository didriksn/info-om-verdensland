# Informasjon om verdensland
- Trykk på et land og se masse ulik informasjon om landet
- Informasjon tatt fra [REST API](https://restcountries.com/#endpoints-full-name) og [Wikimedia REST API](https://en.wikipedia.org/api/rest_v1/#/Page%20content)
- Enkelt informasjon for enkelte land er mulig at det ikke er i API-ene
- Informasjonen tar noen raske sekunder før de laster første gang man trykker på landet
